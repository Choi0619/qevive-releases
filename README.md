# Qevive Releases

**Qevive**는 암호화된 음원을 재생하는 구독형 오디오 플레이어 앱이에요.
이 저장소는 Qevive **앱/데스크톱 프로그램의 설치 파일을 배포**하기 위한 공간이며, 소스 코드는 포함되어 있지 않습니다.

## 다운로드

최신 버전은 [Releases](../../releases/latest) 페이지에서 받을 수 있습니다.

| 플랫폼 | 파일 | 설치 방법 |
|---|---|---|
| Android | `app-release.apk` | 다운로드 후 실행 (출처를 알 수 없는 앱 설치 허용 필요) |
| Windows | `qevive-windows-x.x.x.zip` | 압축 해제 후 `Qevive.exe` 실행 (설치 프로그램 없이 바로 실행되는 포터블 방식) |

## 자동 업데이트

앱을 실행할 때마다 최신 버전인지 자동으로 확인하고, 새 버전이 있으면 앱 안에서 바로 업데이트할 수 있어요. 이 저장소에서 파일을 직접 다시 받을 필요는 없습니다 (최초 설치 시에만 필요).

## 문의
